Imparare a usare Claude Code da zero, una skill alla volta, senza essere programmatori.

Claude Code è l'assistente di Anthropic che vive nel tuo terminale: gli parli in italiano, lui legge i file, scrive codice, esegue comandi e ti spiega ogni passo mentre lo compie. Le skill sono istruzioni riutilizzabili che gli insegnano un mestiere — scrivere bene, progettare un'interfaccia, controllare la sicurezza. Qui le scegliamo per te, una a una, con parole semplici.
